## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Covid-Bookings/Front-End-Covid-Bookings
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
